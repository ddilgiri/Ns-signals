# 📱 Install NSE F&O Signal Engine on Android — NO PC NEEDED

Follow these steps carefully. Takes about 10 minutes.
Everything is free. No PC needed after setup.

═══════════════════════════════════════════════════
PART 1 — Upload files to the internet (free cloud)
═══════════════════════════════════════════════════

We'll use GitHub (free file storage) + Render (free server).

────────────────────────────────────────────
STEP 1 — Create a free GitHub account
────────────────────────────────────────────

GitHub is just a free place to store code files online.

1. Open your phone browser → go to: https://github.com
2. Tap "Sign up"
3. Enter email, password, username
4. Verify your email
5. You now have GitHub ✅

────────────────────────────────────────────
STEP 2 — Create a new repository (folder)
────────────────────────────────────────────

A "repository" is just an online folder for your files.

1. After logging into GitHub, tap the "+" icon (top right)
2. Tap "New repository"
3. Repository name: nse-signals
4. Select: Private (keep your Angel One credentials safe)
5. Tap "Create repository"

────────────────────────────────────────────
STEP 3 — Upload all files from the zip
────────────────────────────────────────────

1. Extract the zip file on your phone (tap zip → Extract)
2. In your new GitHub repository, tap "uploading an existing file"
3. Tap "choose your files"
4. Select ALL files from the extracted folder:
   - server.js
   - index.html
   - package.json
   - package-lock.json
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
   - Procfile
   - render.yaml
5. Tap "Commit changes" (green button)
6. All files are now online ✅

═══════════════════════════════════════════════════
PART 2 — Start the free server on Render.com
═══════════════════════════════════════════════════

Render runs your server.js 24/7 for free in the cloud.

────────────────────────────────────────────
STEP 4 — Create Render account
────────────────────────────────────────────

1. Go to: https://render.com
2. Tap "Get Started for Free"
3. Tap "Sign in with GitHub" (easiest — uses your GitHub account)
4. Authorize Render ✅

────────────────────────────────────────────
STEP 5 — Deploy your server
────────────────────────────────────────────

1. In Render dashboard, tap "New +" → "Web Service"
2. Tap "Connect" next to your "nse-signals" repository
3. Fill in:
   - Name: nse-fno-signals (or anything)
   - Runtime: Node
   - Build Command: npm install
   - Start Command: node server.js
   - Instance Type: Free
4. Tap "Create Web Service"
5. Wait 2-3 minutes while it builds...
6. You'll see a URL like: https://nse-fno-signals.onrender.com ✅

   ⚠️ SAVE THIS URL — you need it in the next step!

═══════════════════════════════════════════════════
PART 3 — Open the app on your Android phone
═══════════════════════════════════════════════════

────────────────────────────────────────────
STEP 6 — Open the app
────────────────────────────────────────────

1. Open Google Chrome on your Android phone
2. Go to your Render URL + /index.html
   Example: https://nse-fno-signals.onrender.com/index.html
3. The app opens! ✅

────────────────────────────────────────────
STEP 7 — Install as home screen app
────────────────────────────────────────────

1. Chrome shows a banner "Add to Home Screen" → Tap Install
   (Or tap Chrome menu ⋮ → "Add to Home screen")
2. Tap "Add"
3. App icon appears on your home screen!
4. Open it — runs full screen, no browser bar 📱✅

────────────────────────────────────────────
STEP 8 — Connect Angel One
────────────────────────────────────────────

1. Open app → tap "Setup" tab (bottom)
2. Server URL is already set automatically ✅
3. Enter your Angel One details:
   - Client ID: your Angel One login ID (e.g. A123456)
   - SmartAPI Key: from smartapi.angelbroking.com (free)
   - Password: your trading password
   - TOTP Secret: optional (for auto 2FA)
4. Tap "Connect Angel One"
5. Go to "Signals" tab → Tap ▶ SCAN
6. Live F&O signals appear! ✅

═══════════════════════════════════════════════════
IMPORTANT NOTES
═══════════════════════════════════════════════════

⚠️  Free Render servers "sleep" after 15 minutes of no use.
    First load after sleeping takes ~30 seconds. This is normal.
    Upgrade to Render's $7/month plan to avoid sleeping.

⚠️  Your Angel One credentials are sent to your Render server.
    Always use a Private GitHub repository (not Public).
    Never share your Render URL publicly.

⚠️  Signals are algorithmic only. Not SEBI-registered advice.
    Always use stop-loss. Paper trade first.

═══════════════════════════════════════════════════
GETTING ANGEL ONE SMARTAPI KEY (FREE)
═══════════════════════════════════════════════════

1. Go to: https://smartapi.angelbroking.com
2. Sign in with your Angel One account
3. Go to "My Apps" → "Create New App"
4. App name: anything (e.g. MySignalApp)
5. Copy the API Key shown ✅

═══════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════

App not loading?
→ Wait 30 seconds (server waking up from sleep)
→ Check your Render URL is correct

"Not connected" error?
→ Go to Setup tab → tap Connect again
→ Check Client ID, API Key, Password are correct

Premiums showing "EST" not "LIVE"?
→ Market must be open: 9:15 AM – 3:30 PM Mon–Fri
→ Try reconnecting Angel One

Render deploy failed?
→ Make sure ALL files were uploaded including package.json
→ Check Render logs for error messages
